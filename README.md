# Cyber SOC Analysis

This project presents a cybersecurity incident investigation involving multiple attack techniques.

## Overview
A monitoring system detected suspicious activities including:
- Multiple failed login attempts
- Successful admin login
- Privilege escalation to root
- Phishing email attack
- User credential compromise

## Attack Types Identified
- Brute-force attack
- Privilege escalation
- Phishing attack
